# SwiftSight Sprint Planning Project

## Overview
SwiftSight 제품의 스프린트 플래닝을 Jira + HTML Gantt 캘린더로 관리하는 프로젝트.

## Key Files
- `sprint-planning.html` — Gantt 차트 캘린더 (개발+QA 일정)
- `.claude/skills/jira-sprint-planner.md` — Jira CRUD 스킬 (카드 생성/수정/링크/상태전환)
